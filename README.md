Sentinel – Intelligent Night Safety Companion
Overview

Sentinel is an intelligent personal safety application designed to enhance user security during night travel and outdoor activities. The application provides real-time safety monitoring, location-based risk assessment, emergency assistance features, and smart safety recommendations to help users stay safe and connected.

The project was developed using modern web technologies and Firebase services to ensure scalability, reliability, and real-time data synchronization.

Continuous user activity monitoring during walks and travel.

Application Workflow
User launches the application.
User authentication is performed.
Safety dashboard is displayed.
Location and safety data are processed.
Safety index is generated.
User receives real-time safety information.
Emergency features remain accessible throughout usage.
User can manage profile and safety preferences.
Objectives

The primary objective of Sentinel is to provide users with a reliable digital safety companion that improves situational awareness and enables faster responses during potentially unsafe situations.

The system aims to:

Improve personal safety.
Increase user confidence during travel.
Provide real-time safety information.
Enable quick emergency communication.
Promote safer mobility practices.
